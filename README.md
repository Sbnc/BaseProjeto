## BASE DO PROJETO

<p>Base do projeto em  node utilizando TS.</p>
<p>Executar os seguintes comandos:</p>

 `yarn` - Instalação das dependências
 `yarn build` - Criação da build
 `yarn start` - Iniciar o projeto
